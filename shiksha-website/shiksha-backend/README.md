## SHIKSHANA BACKEND

## Installation

This is a [Node.js](https://nodejs.org/en/) module available through the
[npm registry](https://www.npmjs.com/).

Before installing, [download and install Node.js](https://nodejs.org/en/download/).
Node.js 16.20.0

Installation steps -

```
npm i
npm i -g nodemon
npm run start-dev

```
